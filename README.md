# 🚀 Cyber Cafe Billing System

A modern, automated management and billing solution for Cyber Cafes. Built with **Python** and **Tkinter**, this system simplifies session tracking, member management, and real-time revenue calculation.

---

## ✨ Features
- **Instant Session Management:** Start and end customer sessions with a single click.
- **Real-time Billing:** Automatically calculates costs based on session duration.
- **Member Database:** Add and manage regular members with persistent data.
- **Modern Neon UI:** Dark-themed, high-contrast interface for better usability.
- **Data Persistence:** Integrated with **SQLite** to ensure all logs and billing history are saved securely.

## 🛠️ Project Structure
- `main.py`: The entry point of the application.
- `gui.py`: Contains the UI design and Tkinter layout logic.
- `database.py`: Handles SQLite connection and table creation.
- `billing.py`: The core logic for calculating bills and tracking time.
- `reports.py`: Generates summaries of sessions and revenue.

## 🚀 Getting Started

### Prerequisites
- Python 3.x
- Tkinter (usually comes pre-installed with Python)

### Installation
1. **Clone the repository:**
   ```bash

2. **Navigate to the directory:**
   cd ACP-Project
   
4. **Run the application:**
   python main.py
